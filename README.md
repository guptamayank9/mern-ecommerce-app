# MERN E-commerce App

This is a full stack e-commerce web application built using MongoDB, Express, React and Node.js.

## Features

* User signup and login (JWT authentication)
* Product listing (Men, Women, Kids)
* Add to cart functionality
* Cart data stored in database
* Increase / decrease quantity
* New collections and popular products section
* Admin can add products with image upload

## Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express
* Database: MongoDB
* Authentication: JWT

## How to run

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev

## Environment Variables

Create a `.env` file in backend:

MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
